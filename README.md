# ADA 442 - Bank Marketing Subscription Prediction

## Project Overview
This project develops a machine learning model to predict whether a client will subscribe to a term deposit. Using Portuguese banking data, the study focuses on optimizing marketing efficiency through predictive analytics.

## Key Features
- **Data:** Client demographics, financial history, and socio-economic indicators.
- **Preprocessing:** Mode imputation, categorical encoding, and feature selection.
- **Modeling:** Comparative analysis of classification algorithms (Random Forest, Logistic Regression, etc.).
- **Evaluation:** The final Random Forest model achieved a **Test ROC AUC of 0.756**.

## Files
- `Ada 442 Project (1).ipynb`: Main analysis and modeling notebook.
- `rf_pipeline.sav`: Saved Random Forest model for inference.

## Quick Start
1. Install dependencies: `pip install pandas scikit-learn seaborn`
2. Run the notebook to see the full pipeline and performance metrics.
